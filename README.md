# WebGPU Sandbox

Experimenting with WebGPU as a cross-platform graphics API

## Build

Use CMake to build

```
mkdir build
cmake -S . -B ./build -G <generator-name>
cmake --build ./build [--config (Debug|Release|RelWithDebInfo)]
```
